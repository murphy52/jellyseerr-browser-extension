How to get your API Key

  1. Open your Jellyseerr web interface
  2. Go to Settings → General
  3. Scroll down to find the API Key section
  4. Copy the API key and paste it above

Features

  • ✅ Add request buttons to IMDB movie and TV show pages
  • ✅ Add request buttons to Rotten Tomatoes movie and TV show pages
  • ✅ Automatic media matching using title and year
  • ✅ Visual feedback for request status
  • ✅ Support for both movies and TV series
